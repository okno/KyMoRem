## Casing Rules

- Use `KyMoRem` for the product name.
- Use `Keyboard Mouse Remote` for the extended name.
- Use `KMR` for favicon/icon text and compact HUD marks.
- Do not use `KYMOREM`, `Kymorem`, `kymorem` or other variants in visible UI
  unless the lowercase form is required for package ids, binary names or paths.
