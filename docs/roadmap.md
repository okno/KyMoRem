# Roadmap

## 0.1 Technical Seed

- Windows host UI and tray.
- Linux X11 client.
- Secure TCP transport.
- Encrypted LAN discovery.
- Optional email relay.
- IT, EN and CH localization slots.

## 0.2 Native Input Expansion

- Windows SendInput backend.
- macOS CGEvent backend.
- X11 XTest backend.
- Capability reporting per backend.

## 0.3 Pairing and Trust Store

- Device fingerprints.
- Pairing confirmation on both endpoints.
- Trust revoke command.
- Token rotation workflow in UI.

## 0.4 Android Device Mode

- Android pairing screen.
- App-local pointer surface.
- Protocol implementation.
- Latency and reconnect handling.

## 0.5 Layout Editor

- Multi-client layout.
- Drag-and-drop screen placement.
- Per-edge routing.
- Visual active-device indicator.

## 0.6 Operational Hardening

- Structured logs.
- Exportable diagnostics bundle.
- Rate limiting.
- Signed release verification.
